# Wiki Ciberseguridad

**Cómo usar**
1. Crea una carpeta `wiki-ciberseguridad` en tu equipo.  
2. Copia cada bloque marcado `FILE: <filename>` en archivos individuales con el mismo nombre y ubicaciones indicadas.  
3. Coloca imágenes en la carpeta `img/`.  
4. Sube el proyecto a GitHub y publica con GitHub Pages, Netlify o Vercel.  

**Recomendaciones**
- Mantén las referencias en `referencias.html` y añade citation dates.  
- Para edición colaborativa, trabajen en Google Docs y peguen contenido final en los HTML.
